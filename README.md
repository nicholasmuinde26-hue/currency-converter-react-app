# 💱 Currency Converter React App

A simple and responsive currency converter built with **React**.  
This app lets users input an amount, select a source currency, and convert it into a target currency (USD, EUR, GBP, JPY).  

---

## 🚀 Features
- Input field for amount
- Dropdowns for selecting source and target currencies
- Real-time conversion using memoized calculations (`useMemo`)
- Clean and responsive UI
- Easy to extend with more currencies or APIs

---

## 🛠️ Tech Stack
- **React** (Hooks: `useState`, `useMemo`)
- **CSS** for styling (responsive card layout)
- JavaScript ES6 features

---

## 📂 Project Structure
# currency-converter-react-app
